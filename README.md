# Goal Tracker App
==>  Goal Tracker App developed using React Native, TypeScript, Expo, and React Navigation. It provides users with the ability to create and track their goals effectively. Leveraging the robust capabilities of React Native, the app delivers a seamless user experience and incorporates various powerful tools to enhance its functionality.

# Run_Locally

Clone the project

```bash
https://github.com/saddamarbaa/goal-tracker-app-react-native-typescript
```

Go to the project directory

```bash
 goal-tracker-app-react-native-typescript
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn start
```
