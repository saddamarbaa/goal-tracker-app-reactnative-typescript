# Run_Locally

Clone the project

```bash
https://github.com/saddamarbaa/react-native-goals-app
```

Go to the project directory

```bash
 react-native-goals-app
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn start
```
